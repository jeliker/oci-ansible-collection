.. Title

oracle.oci.oci_compute_cloud_at_customer_ccc_upgrade_schedule -- Manage a CccUpgradeSchedule resource in Oracle Cloud Infrastructure
++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++

.. Collection note

.. note::
    This plugin is part of the `oracle.oci collection <https://galaxy.ansible.com/oracle/oci>`_ (version 5.5.0).

    You might already have this collection installed if you are using the ``ansible`` package.
    It is not included in ``ansible-core``.
    To check whether it is installed, run :code:`ansible-galaxy collection list`.

    To install it, use: :code:`ansible-galaxy collection install oracle.oci`.

    To use it in a playbook, specify: :code:`oracle.oci.oci_compute_cloud_at_customer_ccc_upgrade_schedule`.

.. contents::
   :local:

.. Deprecated


Synopsis
--------

.. Description

- This module allows the user to create, update and delete a CccUpgradeSchedule resource in Oracle Cloud Infrastructure
- For *state=present*, creates a new Compute Cloud@Customer upgrade schedule.
- This resource has the following action operations in the :ref:`oracle.oci.oci_compute_cloud_at_customer_ccc_upgrade_schedule_actions <ansible_collections.oracle.oci.oci_compute_cloud_at_customer_ccc_upgrade_schedule_actions_module>` module: change_compartment.


.. Aliases


.. Requirements

Requirements
------------
The below requirements are needed on the host that executes this module.

- python >= 3.6
- Python SDK for Oracle Cloud Infrastructure https://oracle-cloud-infrastructure-python-sdk.readthedocs.io
